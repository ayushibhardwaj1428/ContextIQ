# ContextIQ

ContextIQ is a context-aware AI assistant that allows users to upload documents and ask questions grounded in the uploaded context.

## Features

* Upload TXT documents
* Context-aware AI chat
* Gemini-powered responses
* Agent pipeline visualization
* Session-based context storage
* Responsive UI
* Context insights panel

## Tech Stack

* Next.js
* TypeScript
* Zustand
* Material UI
* Google Gemini API

## Demo Flow

1. Upload a document
2. Ask questions about the document
3. ContextIQ retrieves context
4. AI generates grounded responses

## Future Enhancements
